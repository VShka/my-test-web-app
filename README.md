# Тестовое веб-приложение, созданное в процессе обучения

# Стек: 
- Python3
- Фреймворк Flask
- СУБД sqlite
- Bootstrap 4

# Что можно делать?
- Переходить по страницам
- Заполнить форму и оставить комментарий
- Данные записываются в БД (Не оставляйте конфиденциальных данных!!!)

- валидация формы
